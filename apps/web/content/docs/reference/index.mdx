## Quick Links

### Beads Tools

| Tool | Description |
|------|-------------|
| `beads_create` | Create a new bead with type-safe validation |
| `beads_create_epic` | Create epic with subtasks atomically |
| `beads_query` | Query beads with filters |
| `beads_update` | Update bead status/description |
| `beads_close` | Close bead with reason |
| `beads_sync` | Sync beads to git |

### Swarm Tools

| Tool | Description |
|------|-------------|
| `swarm_select_strategy` | Analyze task and recommend decomposition strategy |
| `swarm_decompose` | Generate decomposition prompt |
| `swarm_spawn_subtask` | Prepare subtask for parallel execution |
| `swarm_complete` | Mark subtask complete, release reservations |
| `swarm_status` | Get swarm progress by epic ID |

### Agent Mail Tools

| Tool | Description |
|------|-------------|
| `swarmmail_init` | Initialize agent session |
| `swarmmail_send` | Send message to agents |
| `swarmmail_inbox` | Fetch inbox (context-safe) |
| `swarmmail_reserve` | Reserve files for exclusive edit |
| `swarmmail_release` | Release file reservations |

### Effect-TS Primitives

| Primitive | Description |
|-----------|-------------|
| `DurableCursor` | Checkpointed event stream consumption |
| `DurableLock` | Distributed mutual exclusion via CAS |
| `DurableMailbox` | Actor-style message passing |
| `DurableDeferred` | Distributed promises |
